📖 Mailcow-Setup-Anleitung anzeigen

Idee: Eine Subdomain (z.B. mt.dmarc-geeks.ch) bekommt eine Catch-All-Mailbox in Mailcow. Die App pollt die alle paar Sekunden, matched mt-{token}@mt.dmarc-geeks.ch auf offene Tests, scored die Mail.

  1. MX-Record für mt.dmarc-geeks.ch auf deinen Mailcow-Host setzen.
  2. In Mailcow: Configuration → Mail Setup → Domains → Add Domain → mt.dmarc-geeks.ch (Mailbox-Quota 50 MB reicht).
  3. Mailboxes → Add Mailbox: catch-all@mt.dmarc-geeks.ch, Passwort kopieren.
  4. Aliases → Add Alias: Alias = @mt.dmarc-geeks.ch (leerer Local-Part = Catch-All), Goto = catch-all@mt.dmarc-geeks.ch, Active ✓.
  5. Hier unten in der Mail-Tester-Gruppe ENV setzen:
    • MAILTEST_DOMAIN = mt.dmarc-geeks.ch
    • MAILTEST_IMAP_HOST = mail.dmarc-geeks.ch (Mailcow)
    • MAILTEST_IMAP_USER = catch-all@mt.dmarc-geeks.ch
    • MAILTEST_IMAP_PASSWORD = das Passwort aus Schritt 3
    • MAILTEST_IMAP_PORT = 993 · SSL = on
  6. App neu starten (oder „💾 Speichern" + 15s warten).
  7. Hier oben „🔄 Catch-All jetzt einmal pollen" klicken → Resultat zeigt {"status": "ok"}.
  8. /mailtest öffnen → Test-Adresse anzeigen → Mail dahin schicken → Score erscheint.

Wenn's nicht klappt: Mailcow → Logs → Postfix öffnen — siehst du da die Test-Mail eingehen? Wenn nein, ist der MX-Record falsch. Wenn ja aber kein Score: Poll-Button hier oben drücken + Container-Logs der App ansehen.
