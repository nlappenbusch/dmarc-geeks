📦 Mail-Tester einbetten

Embed das Mail-Tester-Tool als iframe auf deiner Website. Kostenlos, gratis Lead-Magnet — gut für IT-Dienstleister, Agenturen, Verbände, MSPs.

👁 Vorschau

So sieht's bei deinen Besuchern aus:

🔗 Embed-Code

Diesen HTML-Code an die gewünschte Stelle auf deiner Website kopieren:

<iframe src="{{ base_url }}/embed/mailtest"
        width="100%" height="520"
        style="border: 0; border-radius: 12px;"
        loading="lazy"
        title="Mail-Tester von DMARC Geeks">
</iframe>

🔍 Mit Tracking-Parameter (optional)

Wenn du verschiedene Embed-Stellen unterscheiden willst (z.B. Homepage vs. Blog-Post), füge ein ?utm-Parameter dran:

<iframe src="{{ base_url }}/embed/mailtest?utm=utm_source=dein-domain&amp;utm_medium=blog"
        width="100%" height="520"
        style="border: 0; border-radius: 12px;"
        loading="lazy">
</iframe>

📡 Events empfangen (JavaScript)

Wenn du auf Test-Events reagieren willst (z.B. Analytics tracken, eigenes „Danke"-Modal zeigen): 

<script>
window.addEventListener('message', (ev) => {
  if (ev.data?.source !== 'dmarc-geeks-embed') return;
  switch (ev.data.type) {
    case 'ready':
      console.log('Mail-Tester geladen');
      break;
    case 'test-started':
      console.log('User hat Test gestartet:', ev.data.token);
      // z.B. Plausible / GA Event tracken
      if (window.plausible) plausible('Mail-Tester Started');
      break;
    case 'test-ready':
      console.log('Score eingetroffen:', ev.data.score);
      if (window.plausible) plausible('Mail-Tester Done', { props: { score: ev.data.score } });
      break;
  }
});
</script>

🧩 Platform-spezifische Snippets

📝 WordPress

In den Gutenberg-Editor → Custom-HTML-Block → iframe einfügen. Oder im Classic-Editor → Text-Tab.

🛍 Shopify / Webflow

Custom Liquid (Shopify) oder Embed-Element (Webflow) → iframe-Code reinpacken. Höhe ggf. anpassen je nach Theme.

⚛️ React / Next.js

iframe als JSX einfügen — selfClosing-Syntax, className statt class. Event-Listener via useEffect.
