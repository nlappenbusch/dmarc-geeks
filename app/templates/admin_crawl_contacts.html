KMU-Kontakt-Crawler — Email + Telefon + Firmenname aus Domain-Listen

Du gibst eine Liste von Domains rein. Wir crawlen pro Domain die typischen Public-Pages (/, /kontakt, /impressum, /team, /about) und extrahieren Emails, CH-Telefonnummern und Firmenname. Output als CSV, direkt verwendbar als Input fürs Batch-Snapshot-Tool.

📋 Was wir tun: Public HTML lesen, User-Agent „DMARCGeeksContactBot", robots.txt respektieren, 1 req/s pro Domain. Impressum-Daten sind in CH per Gesetz öffentlich. Keine Login-Walls, kein Scraping geschützter APIs.

💡 Wo bekommst du Domain-Listen her?

  • Moneyhouse — Branche eingrenzen (Treuhand / IT / Healthcare), CSV-Export
  • Zefix (Handelsregister) — by location filtern, Domain ist oft im Eintrag
  • LinkedIn Sales Navigator — 1-Monat-Trial → Search by Industry → Domain-Liste exportieren
  • Branchenverzeichnis.ch / local.ch — KMU mit Website
  • Google-Maps Filterung („IT-Beratung Zürich") + Tools wie OutScraper
  • Open-Data-CH — opendata.swiss hat Firmen-Datasets

Tipp: Pro Run eine Nische — z.B. „Treuhand-Büros Kt. Zürich 5-20 MA". Saubere Targets > viele.

🔄 Pipeline-Workflow

  1. Hier: Domains → Crawler → CSV mit Emails+Firma
  2. /admin/batch-snapshot: CSV → DNS-Check → Grade A-F + Cold-Mail-Drafts
  3. Cold-Mails kopieren → in Outlook senden
  4. Leads landen in /admin/leads (wenn „persist" gewählt)
  5. Nachfass-Calls + Conversion-Tracking direkt im Lead-Detail
